Pour GitHub Pages :
1. Créer un dépôt public.
2. Envoyer index.html et le dossier assets à la racine.
3. Activer Pages depuis Settings > Pages.
4. Choisir la branche main et le dossier root.
5. Copier l'adresse publique donnée par GitHub dans les descriptions YouTube.

Pour ajouter des vrais liens de téléchargement plus tard :
- Remplacer les liens des cartes projet par les URLs finales des vidéos, posts Discord, pages Ko-fi, releases GitHub ou fichiers hébergés.
- Ne pas héberger de gros ISO dans le dépôt GitHub Pages.
- Garder GitHub Pages comme vitrine et stocker les fichiers lourds ailleurs.
